Chapter 1: Structure

The structure of an HTML document includes the use of elements such as <html>, <head>, and <body>. These elements help to define the layout and content of a web page.

Chapter 3: Lists

Ordered List

  1. First item
    1. Sub item 1
    2. Sub item 2
  2. Second item
    1. Sub item 1
    2. Sub item 2
      1. Sub sub item 1
      2. Sub sub item 2
  3. Third item

Unordered List

  • First item
    • Sub item 1
    • Sub item 2
  • Second item
    • Sub item 1
    • Sub item 2
      • Sub sub item 1
      • Sub sub item 2
  • Third item

Definition List

HTML
A markup language for creating web pages.
CSS
A style sheet language used for describing the presentation of a document written in HTML.
JavaScript
A programming language commonly used in web development to create interactive effects within web browsers.
